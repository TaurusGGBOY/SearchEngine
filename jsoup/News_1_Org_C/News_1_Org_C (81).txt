MySQL 是最流行的关系型数据库管理系统，在 WEB 应用方面 MySQL 是最好的 RDBMS(Relational Database Management System：关系数据库管理系统)应用软件之一。 略过MySQL 的安装部署，直接学习如何使用MySQL数据库 命令提示符连接：mysql -u root -p CREATE DATABASE 数据库名; 例如：CREATE DATABASE mybatis; DELETE DATABASE 数据库名；例如：DELETE DATABASE mybatis； 连接上MySQL后，选择数据：USE 数据库名，例如：use RUNOOB; MySQL支持所有标准SQL数值数据类型。 语法：CREATE TABLE table_name (column_name column_type); 语法：DROP TABLE table_name ; 1、当你不再需要该表时， 用 drop; 2、当你仍要保留该表，但要删除所有记录时， 用 truncate; 3、当你要删除部分记录时， 用 delete； 语法：INSERT INTO table_name ( field1, field2,...fieldN ) VALUES ( value1, value2,...valueN ); 如果所有的列都要添加数据可以不规定列进行添加数据，例如： INSERT INTO runoob_tbl VALUES (0, "JAVA 教程", "RUNOOB.COM", '2016-05-06'); 语法：SELECT column_name,column_name FROM table_name [WHERE Clause] [LIMIT N][ OFFSET M] 以上是mysql数据库的最最基本的sql语法，后面会继续整理。
